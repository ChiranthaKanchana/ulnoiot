wemos d1 mini - rfid-rc522 board

3v3 - 3.3V
d8  - sda
d7  - MOSI
d6  - MISO
d5  - SCK
d0  - RST
G   - GND

Add device with:
r=d("mfrc522","r",d0)

Write data to a mifare classic card:
r.write("mydata to write to reader")

nodered code for toggle function:
if(msg.payload === "on")
{
    if(context.togglestate === "on")
    {
    context.togglestate = "off";
    }
    else{
        context.togglestate ="on";
    }
